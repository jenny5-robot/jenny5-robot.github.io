Jenny 5 robot - troubleshooting

  • Cannot connect to the robot from Android app

    Check the followings:

    • Check if the server application is running on the Jenny 5 computer.

    • Check if the Jenny 5 computer is connected to the hotspot of the phone controlling it.

    • Check the IP of the Jenny 5 computer (run ipconfig). If the phone and the Jenny 5 are on different networks, make sure that the router has the port forwarding correctly.

    Robot does not move

    • Make sure that batteries are turned on.

    • Check if server application is running.

    • Check USB port numbers from Device Manager.
